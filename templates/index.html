  Design Condition
Fully deteriorated condition is designed to support hydraulic, soil, and live loads. The groundwater level, soil type and depth, and live load should be determined by the engineer.

Partially deteriorated condition is designed to support the hydraulic loads due to groundwater, since the soil and surcharge loads can be supported by the original pipe. The soil adjacent to the existing pipe must provide adequate side support. The pipe may have longitudinal cracks and up to 10.0% distortion of the diameter.

{% if error %}{% if error["location"] %} {% else %} {% endif %}{% else %} {% endif %}

  Pipe Location
Use this option to apply standard AASHTO HS-20 Highway, Cooper E-80 railroad, or 180 kip airplane gear assembly live loads. The values of the live load pressures are calculated with respect to depth and include an impact factor of 1.5 to account for bumps and irregularities in the travelled surface.

See American Lifelines Alliance for additional details.

  CIPP Design Modulus of Elasticity
Enter the modulus of elasticity in psi from the manufacturer. ASTM F1216 requires a minimum value of 250,000 psi as tested by ASTM D790. NASSCO recommends using 400,000 psi for enhanced composites.

  CIPP Design Flexural Strength
Enter the design flexural strength in psi as provided by the manufactureer. ASTM F1216 has set a minimum value for field inspection of 4,500 psi as tested by ASTM D790. NASSCO recommends using 4,500 psi for standard and enhanced composites.

  Enhancement Factor
The enhancement factor (K) is used for design in partially deteriorated conditions. ASTM recommends a minimum value of 7.0 where there is full support of the existing pipe. NASSCO designates that 7.0 shall be used for partially deteriorated design.
